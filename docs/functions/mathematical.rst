Mathematical Operators and Functions
====================================

Operators
---------

Mathematical operators are provided for many Drizzle types.

========    ==================   ============  ======
Operator    Description          Example       Result
========    ==================   ============  ======
``+``       addition             ``2 + 3``     5
``-``       subtraction          ``2 - 3``     -1
``*``       multiplication       ``2 * 3``     6
``/``       division             ``4 / 2``     2
``%``       modulo (remainder)   ``5 % 4``     1
``|/``      square root          ``|/ 25.0``   5
``||/``     cube root            ``||/ 27.0``  3
``!``       factorial            ``5 !``       120
``!!``      factorial            ``!! 5``      120
``@``       absolute value       ``@ -5.0``    5
========    ==================   ============  ======
